# backend

# 개요
대학생의, 대학생에 의한, 대학생을 위한 시험지 제작 웹

전공,교양,시사 등을 공부 하기 위해 대학생을 위한 시험지 제작 웹 사이트입니다.

분야에 맞는 다양한 문제들로 구성된 시험지를 제공하며, 인기있는 시험지 또한 분류하여 보여줍니다.

서적에 해당되는 문제에만 국한되지 않고, 대학 별로 다양한 문제를 접할 수 있습니다.
